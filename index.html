Blossom a language built to grow

Blossom is a functional programming language that combines the speed and power of Haskell with the simplicity and learnability of Python.

Blossom, like this website, is still a work in progress. We the authors reserve the right to make any changes we see fit to the language specification at this time. However we'd love for you to start using the language and to give us feedback on how we can improve it. A somewhat functional Blossom interpretor can be found in our our git repo.
