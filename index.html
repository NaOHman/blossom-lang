Blossom a language built to grow

Blossom is a functional programming language that combines the speed and power of Haskell with the simplicity and learnability of Python.

Blossom, like this website, is still a work in progress. We the authors reserve the right to make any changes we see fit to the language specification at this time. However we'd love for you to start using the language and to give us feedback on how we can improve it. A somewhat functional Blossom interpreter can be found in our our git repo.

Features

Message Passing

Blossom contains first class support for message passing which highlights side effects, promotes modular design, and helps developers write massively parallel applications for the modern age.

High Level Abstraction

Blossom supports function overloading, parametric polymorphism, and inheritance so you can write fewer lines of code that do more.

Strong Static Types

Data in Blossom is typed with a variant of the Hindley Milner type system that provides strong static guarantees. Best of all Types are completely inferable so you don't have to write any type signatures.

Compilation

Blossom (will be) a compiled language meaning that it will on average run faster than Python, Javascript, and even Java.

Garabage Collection

Blossom provides automatic garbage collection via compiler optimization so memory leaks are a thing of the past.

Hotplugging

Blossom code can be hotplugged so that code can be changed while an application is running which means zero downtime when upgrading your apps.
